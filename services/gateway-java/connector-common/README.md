# Module: connector-common

## 1. 模块领域
本模块定义了网关系统最基础的 **跨语言通讯协议 (Protocol)** 和 **通用 DTO**。它是所有模块（包括服务端和 SDK）的共同依赖。

## 2. 能力范围
- 定义 `EventFrame`（数据帧）等协议模型。
- 定义 `AcquisitionResult` 等通用枚举。
- 提供不依赖任何框架（如 Spring）的基础工具类（如计算签名、字符串处理）。

## 3. 准入规范
- **适合加入**: 核心通讯模型、轻量级常量定义、纯算法工具类。
- **严禁加入**: 带有 `IP` 地址、数据库连接、Spring Bean 或任何业务编排逻辑的代码。
