# 非功能性需求 (NFRs)

## 1. 性能指标 (Performance)
- **转发延迟 (Latency)**：
  - 内存/Redis 缓存模式下：P99 < 10ms。
  - 数据库模式下：P99 < 50ms。
- **并发能力 (Throughput)**：
  - 单实例支持 200 QPS。
  - 分布式集群模式支持 500+ QPS。

## 2. 可靠性与弹性 (Resilience)
- **超时控制 (Timeout)**：
  - 存储操作超时：3s。
  - 外部 API 转发超时：30s。
- **重试机制 (Retry)**：
  - Token 刷新失败重试：指数退避 (1s, 2s, 4s)，最大 3 次。
- **数据库连接池 (Connection Pool)**：
  - 默认最小连接数：2。
  - 默认最大连接数：20。

## 3. 分布式约束
- **可用性保障**: 分布式环境下，各节点需通过数据库连接心跳维持存储可用性。
