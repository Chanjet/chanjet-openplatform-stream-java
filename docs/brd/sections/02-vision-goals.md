# 商业目标与愿景 (Vision & Goals)

## 项目愿景
将 Cowen 打造为开放平台生态中事实上的 **标准 ISV 边车**，通过解耦 Token 管理、安全代理与业务逻辑，使 ISV 能够以“零配置、自愈式、可横向扩展”的方式快速上线高质量应用。

## 商业目标
- **生产化转型**：实现从单机 PC 工具到分布式云原生边车的跨越，支持 100% 的生产部署场景。
- **生态渗透**：通过支持“自建应用”与“商店应用”双模式，覆盖 90% 以上的 ISV 应用类型。
- **性能卓越**：通过引入 Redis 缓存与数据库持久化，显著提升 Token 刷新与请求转发的吞吐量，降低延迟。

## 核心业务诉求
- **商店应用长效授权保障**：建立完善的 `userToken` 与 `orgToken` 维护机制。在用户未取消授权的前提下，系统应具备自动“召回”新鲜 Token 的能力，消除因 Token 过期导致的业务中断。
- **持久化授权回溯能力**：支持通过“用户永久授权码”与“企业永久授权码”换取 Token 的方式，确保 Sidecar 在极端场景（如长期离线重启、凭据丢失）下能自主恢复授权，实现无感自愈。
