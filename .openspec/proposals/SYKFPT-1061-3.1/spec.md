# Spec: Message Dispatching Requirements (SYKFPT-1061-3.1)

## 1. 性能规范
- **分发损耗**: 本地分发逻辑（不含网络 I/O）必须在 5ms 内完成。
- **并发能力**: 支持同时处理 1000+ 并发 Webhook Dispatch 请求（基于 Java 21 虚拟线程）。

## 2. 可靠性规范
- **快速失败**: 如果 `IRouteStore` 连接失败，分发逻辑必须立即返回 503 错误，不得挂起线程。
- **状态一致性**: 分发逻辑本身不持有任何中间状态，完全依赖 SPI 接口。

## 3. 测试覆盖率要求
- `connector-core` 模块的指令覆盖率必须 > 90%。
- 核心分发逻辑的分支覆盖率必须达到 100%。
