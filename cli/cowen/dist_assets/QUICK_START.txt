================================================================================
🚀 cowen CLI 快速上手指南 (v0.3.5)
================================================================================

cowen 是畅捷通开放平台官方提供的全流程治理工具，集成了安全托管、API 调用、流式消息桥接以及基于 AI 向量引擎的智能化接口检索功能。

--------------------------------------------------------------------------------
1. 安装与授权 (Setup & Auth)
--------------------------------------------------------------------------------
- macOS: 双击运行下载的 cowen-v0.3.5-macos-aarch64.pkg (或 x86_64 版本) 文件安装，按照向导完成配置。
- Windows: 双击运行 cowen-v0.3.5-windows-x86_64-setup.exe 安装向导。
- Linux: 
  首先新建目录并解压您下载的 .tar.gz 安装包，然后执行安装脚本：
  $ mkdir cowen_install && tar -xzf cowen-v0.3.5-linux-x86_64.tar.gz -C cowen_install
  $ cd cowen_install && ./install.sh

安装完成后，打开新终端，执行全局环境初始化：
$ cowen init

（CLI 会引导您录入 AppKey 等核心凭据或进行 OAuth2 授权，您的安全信息将被加密存储在本地的 Vault 中）

--------------------------------------------------------------------------------
2. 调用 API 与 AI 智能化检索 (API & AI Search)
--------------------------------------------------------------------------------
查找并搜索可用的 API（支持自然语言意图搜索，需开启 AI 插件）：
$ cowen api list --search "帮我查一下企业的账套信息"

直接发起 API 请求（系统会自动处理签名与 Token）：
$ cowen api GET /v1/user/info

--------------------------------------------------------------------------------
3. 开启实时消息桥接 (Streaming Bridge)
--------------------------------------------------------------------------------
若需接收云端消息推送并自动转发至您本地的 Webhook 服务：
$ cowen daemon start --proxy-port 8080 --foreground

如果已经将 Daemon 注册为系统级后台服务，可以直接通过以下命令管理：
$ cowen daemon restart
$ cowen daemon stop

--------------------------------------------------------------------------------
4. 状态检查与扩展管理 (Maintenance & Plugins)
--------------------------------------------------------------------------------
查看当前登录环境与全局状态：
$ cowen system status

管理或启用本地功能扩展插件（例如开启/关闭本地 AI 模型）：
$ cowen plugins list
$ cowen plugins enable libcowen_search_embedding

--------------------------------------------------------------------------------
5. 进阶使用与多模式指南 (Advanced Usage)
--------------------------------------------------------------------------------
CLI 支持多种应用接入模式与高级运维特性，详细信息请查阅您的安装包（或源码）中附带的 `docs/usage` 目录文档：
- OAuth2 授权接入：查阅 `oauth2.md`
- 自建应用接入 (基于 AppKey/AppSecret)：查阅 `self_built.md`
- 商店应用接入：查阅 `store_app.md`
- 高级配置、健康检查 (cowen doctor) 与运维指标监控：查阅 `OPERATIONS.md`

--------------------------------------------------------------------------------
如有疑问，请运行 'cowen --help' 或查阅畅捷通开放平台官网文档。
