================================================================================
🚀 owenc CLI 快速上手指南 (v0.1.2)
================================================================================

owenc 是畅捷通开放平台官方提供的全流程治理工具，集成了安全托管、API 调用与流式消息桥接功能。

--------------------------------------------------------------------------------
1. 安装与配置 (Setup)
--------------------------------------------------------------------------------
- macOS: 双击 .pkg 文件安装。
- Windows: 右键运行 install.ps1 脚本。
- Linux: 运行 sudo ./install.sh 脚本。

安装完成后，打开新终端执行：
$ owenc init

根据提示输入您的 AppKey, AppSecret 和 Certificate。这些信息将被加密存储在本地安全存储(Vault)中。

--------------------------------------------------------------------------------
2. 开启实时消息桥接 (Streaming Bridge)
--------------------------------------------------------------------------------
若需接收云端消息推送并转发至本地 Webhook：
$ owenc daemon start --proxy-port 8080

查看运行状态：
$ owenc status

--------------------------------------------------------------------------------
3. 调用 API (API Call)
--------------------------------------------------------------------------------
直接发起 API 请求（自动处理签名与授权）：
$ owenc api GET /v1/user/info

查找并搜索可用的 API：
$ owenc api list --search "查询账套"

--------------------------------------------------------------------------------
4. 日志与运维 (Maintenance)
--------------------------------------------------------------------------------
实时查看审计或系统日志：
$ owenc log view audit
$ owenc log view sys

重启后台进程：
$ owenc daemon restart

--------------------------------------------------------------------------------
如有疑问，请访问畅捷通开放平台官网或运行 'owenc --help' 查看详细说明。
