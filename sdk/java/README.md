# Module: connector-sdk-java

## 1. 模块领域
本模块为提供给 ISV 使用的 **客户端开发工具包 (SDK)**。它屏蔽了 WebSocket 通讯细节，提供声明式的编程模型。

## 2. 能力范围
- 自动管理连接：断线重连、指数退避算法、自动心跳 (Ping/Pong)。
- 安全集成：内置签名计算逻辑，自动处理 Nonce 获取。
- 业务回调：支持注册自定义 `EventHandler` 处理接收到的事件。
- 自动 ACK：消息处理成功后自动向网关发送回执。

## 3. 准入规范
- **适合加入**: 客户端 API、重连策略逻辑、面向用户的配置 Builder。
- **严禁加入**: 任何网关服务端的逻辑代码、对 Redis 或微服务注册中心的直接依赖。SDK 必须保持极简，仅依赖基础 HTTP/WebSocket 客户端库。
